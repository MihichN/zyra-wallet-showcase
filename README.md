# ZyraWallet Showcase

Public-safe showcase of a web wallet prototype in the ZyraPayments ecosystem. The private version explores wallet dashboard, address display, balance views, and payment-related user flows.

## Highlights

- Wallet-oriented dashboard UI.
- Balance and address presentation.
- QR/address display flows in the private implementation.
- Foundation for future wallet/payment features.

## Tech Stack

- Next.js, React, TypeScript.
- MySQL-backed data in the private implementation.
- Tailwind CSS, Radix UI.

## Included

- `docs/architecture.md` - wallet UI structure.
- `examples/balance-format/formatBalance.ts` - safe balance formatting example.

## Not Included

Seed phrases, private keys, wallet exports, real addresses, database credentials, and blockchain signing logic are excluded.
